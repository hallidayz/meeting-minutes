# AI GUARDIAN Brand Standards

## Product Identity

| Element | Value |
|---------|-------|
| Product name | **AI GUARDIAN** |
| Short name | AI Guardian |
| Primary tagline | ADVANCING INTELLIGENCE |
| Secondary tagline | PROTECTING HUMANITY |
| App identifier | `com.aiguardian.desktop` |

## Colors

| Color | HEX | Use |
|-------|-----|-----|
| Navy | `#02295b` | Primary text, buttons, dark surfaces |
| Orange | `#fda700` | Accents, active nav, secondary CTAs |
| Light gray | `#d6d6d6` | App shell background |

## CSS Tokens

```css
:root {
  --brand-primary: #02295b;
  --brand-accent: #fda700;
  --brand-background: #d6d6d6;
}
```

## Logo

- Master: `frontend/public/brand/logo.png`
- Tauri icons: `frontend/src-tauri/icons/`

## Do

- Use all-caps **AI GUARDIAN** in marketing and about screens
- Pair taglines: *Advancing Intelligence. Protecting Humanity.*
- Use navy primary buttons with white text

## Don't

- Use retired Acaiguardian purple (`#3740ff`)
- Use Meetily naming in user-facing copy
